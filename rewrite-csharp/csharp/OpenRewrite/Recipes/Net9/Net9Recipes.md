# .NET 9 Breaking Changes Recipes

## Recipe Types

### Search Recipes
Flag affected code with `/* todo: ... */` comments for developer review. Used when automatic refactoring is not feasible because the correct fix depends on context, multiple valid migration paths exist, or behavior changed without an API surface change.

## Status Legend

- **Search** — Search recipe implemented (flags with todo comment for developer review)
- **Skipped** — Not implementable as a C# recipe (with explanation)
