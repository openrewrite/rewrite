`maven-resolution-plan/PHASE-0-SPEC.md` §5. `class` ∈ `ALIGN_TO_MAVEN` (default for anything
semantic) | `KEEP_REWRITE` (output shape/API/seams only — each row must justify itself against
DESIGN.md §0). Every mask in `rewrite-maven/src/test/resources/parity/masks.txt` must reference
a row here.
